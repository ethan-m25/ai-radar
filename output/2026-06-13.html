美国出口管制指令强制停用 Fable 5 —— 外国籍用户全部被切断

不是传言，不是谣言：美国出口管制局（BIS）周五下午 5:21pm ET 给 Anthropic 发了一纸公文，以国家安全为由要求“暂停所有外国国籍用户（包括 Anthropic 外籍员工）对 Fable 5 和 Mythos 5 的访问”。Anthropic 连夜执行了关停——Simon Willison 的 API 访问在 9:59pm ET 精准切断。这是 AI 领域第一次针对“模型出口”的强制行政命令。你的 Fable 5 API 权限很可能已经失效了。

📍 背景

此前两天，Anthropic 刚回滚了 Fable 5 的静默干预政策（Day 3）；Simon Willison 发布了 Fable 5 深度体验（Day 2）。今天的事件是一个突然升级：美国政府援引出口管制法规（EAR），认为 Fable 5 的“绕过破解方法”可能被外国政府用于漏洞挖掘。Anthropic 回应称该破解方案（“让模型读一个代码库并修复漏洞”）实际上 GPT-5.5 也能做到，但政府不予理会。这是 AI 模型第一次被当作“军需品”直接限制出口。

为什么这事重要

  • 你的 Fable 5 访问已经断了 — 如果你是外国国籍（包括中国籍），无论你在美国还是海外，你的 Anthropic API key 调用的 claude-fable-5 现在已经返回 404。Claude.ai 上的 Fable 5 对话、Claude Code 中的 --model claude-fable-5 都将不可用。Simon Willison 在博客中确认了他的 API 在 6/12 晚被切断。
  • 对其他人也成立：这是美国第一次用 AI 模型搞出口管制。 之前出口管制只针对芯片（英伟达 H100 不能卖给中国）。现在扩展到了 AI 模型本身。这意味着未来任何一个“被认为有国家安全风险”的模型都可能被一刀切封禁。
  • 对中文圈的意义 — 如果你用 Anthropic API 做 vibe coding（Claude Code / 自建 agent），Fable 5 的停用会直接影响你的工作流。你需要回退到 Opus 4.8 或使用其他模型（如 DeepSeek V4 Pro、Kimi K2.7-Code、MiMo）。
  • Anthropic 反应非常快 — 从收到指令到完全关停只用了不到 5 小时，说明这是一次重大合规事件。Anthropic 正在与美国政府沟通，但何时恢复不确定。
🎯 今天就能做的事
  1. 确认你的 Fable 5 API 是否还可访问 — 在终端运行：curl -s https://api.anthropic.com/v1/models/claude-fable-5 -H "x-api-key: $(echo $ANTHROPIC_API_KEY)" | jq .。如果返回 404，说明已被阻断。
  2. 切换回 Opus 4.8 或 Sonnet — 如果你用 Claude Code，运行 claude --model claude-opus-4-8。注意 Opus 4.8 的上下文窗口是 200K tokens（Fable 5 是 1M），但日常编码完全够用。
  3. 评估其他模型替代品 — 对比 DeepSeek V4 Pro（开源，862B 参数，中国可用）、Kimi K2.7-Code（1.1T MoE，Moonshot 出品）、MiMo（小米开源，554B MoE）。虽然这些模型的“主动程度”不如 Fable 5，但对于 vibe coding 可能更可靠。
  4. 关注 Anthropic 的后续声明 — Anthropic 官方公告表示“政府在 5:21pm 发来指令，我们必须在通过合法流程之前执行”。后续可能会有法律挑战。

为什么政府突然出手？ — Anthropic 在公开声明中解释：政府认为有人发现了一种“绕过 Fable 5 防护”的方法，可以用来发现软件漏洞。但 Anthropic 回应说：GPT-5.5 也能做到同样的事，这不是 Fable 5 独有的能力。真正的矛盾可能是：Fable 5 的“过度主动”让它在漏洞挖掘方面特别高效，而美国政府认为这种高效能力可能被外国用于攻击美国关键基础设施。

对你意味着什么？ — 作为中国籍用户，你的 Fable 5 访问权已被吊销。但好消息是：1）Anthropic 的其他模型（Opus 4.8、Sonnet 4.6）不受影响；2）中国本土的替代模型越来越强（DeepSeek V4 Pro 在许多基准上已经超越 GPT-5.5）；3）这个管制可能有反效果——它可能加速中国用户的“去 Anthropic 化”。

历史对比 — 这是美国第一次对商业化 AI 模型实施出口管制。之前英伟达的 GPU 出口管制（2022 年）一度造成中国 AI 训练基建中断，但也刺激了华为 Ascend 和寒武纪的加速发展。这次模型出口管制可能产生类似效果：加速中国本土模型生态的成熟。

🔭 后续关注什么

1）Anthropic 是否会对美国政府提出法律挑战；2）管制是临时（几天/几周）还是长期（几个月）；3）Opus 4.8 是否会被后续扩展指令波及；4）OpenAI 的 Fable 5 类似的模型（是否有类似管制风险）；5）中国本土模型的反应速度——DeepSeek 和 Moonshot 可能会借机抢夺用户。
