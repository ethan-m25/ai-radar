DeepSeek V4 Pro — 开源模型精度超越 GPT-5.5 Pro

独立评测网站 runtimewire 发布报告称 DeepSeek V4 Pro 在多项精度基准上超过了 OpenAI 的 GPT-5.5 Pro。如果属实，这意味着开源模型第一次在纯精度上追平甚至反超闭源旗舰。

📍 背景

DeepSeek V4 系列是深度求索（DeepSeek）的第四代模型，V4 Pro 是其中的大尺寸版本（862B 参数，MoE 架构）。GPT-5.5 Pro 是 OpenAI 目前最强的推理模型。过去一年，开源模型在性价比上逐渐逼近闭源，但在顶尖精度上始终有差距。如果这次评测站得住，格局会变。

为什么这事重要

  • 精度反超是里程碑 — 之前开源模型靠便宜和开放吸引人，但精度总差一截。如果 DeepSeek V4 Pro 真的在数学、推理、代码等核心任务上超过 GPT-5.5 Pro，那选择模型时就不必再妥协了。
  • 对 clawii 的实际影响 — 你用的 Claude Code 和 Codex 背后是闭源模型。如果 DeepSeek V4 Pro 精度相当甚至更好，且 API 价格更低（DeepSeek 通常比 OpenAI 便宜 5-10 倍），你可以考虑切换后端模型来省钱。
  • 开源生态的胜利 — 模型权重完全开放，可以在自己的服务器上跑（虽然需要很多 GPU），也可以微调。这对企业级应用意义重大。
🎯 今天就能做的事
  1. 读评测原文 — 去 runtimewire.com 看完整报告，注意他们测试了哪些基准（AIME、GPQA、SWE-bench 等），以及测试方法是否公平。
  2. 试 API — 在 platform.deepseek.com 注册账号，充值几块钱，用 deepseek-v4-pro 模型跑一个你平时用 GPT-5.5 做的任务，对比结果和成本。
  3. 在 Codex 中切换模型 — 如果你用 Codex CLI，可以通过环境变量 OPENAI_API_BASE 指向 DeepSeek 的 API 端点（https://api.deepseek.com/v1），然后设置 OPENAI_API_KEY 为 DeepSeek 的 key。试试看效果。
  4. 关注社区验证 — 去 HN 帖子下面看有没有人复现评测结果。如果社区验证通过，这个信号会升级。

跑在什么之上 — DeepSeek V4 Pro 可以在 HuggingFace 下载权重，支持 vLLM、SGLang、Ollama 等推理框架。API 服务由 DeepSeek 官方提供，也通过 OpenRouter、Together AI 等第三方平台可用。技术栈是 PyTorch + 自定义 MoE 实现。

上手难度 — 用 API 最简单：注册、充值、调用，5 分钟。自托管需要至少 8 张 H100（80GB）才能跑 55B 激活参数，不适合个人。用第三方平台（如 OpenRouter）可以零配置使用。

诚实说缺点 — 评测来自第三方网站，尚未被广泛复现。DeepSeek 模型的“思考”风格可能和 GPT 不同，需要适应 prompt。API 稳定性不如 OpenAI（偶尔有延迟）。中文能力很强，但英文可能略逊于 GPT-5.5 Pro 的某些方面。

你能拿它做什么 — 如果你用 Codex 或 Claude Code 做 vibe coding，可以把后端模型换成 DeepSeek V4 Pro，成本降到原来的 1/5 到 1/10。对于写报告、分析数据、生成代码等任务，效果可能差不多甚至更好。

Lathe — 用 LLM 系统学习新领域，而不是跳过

Show HN 上出现了一个新工具 Lathe，它利用 LLM 生成结构化的学习路径、问题和练习，帮助你真正学会一个新领域，而不是让 AI 替你完成任务。324 分登上 HN，社区反响热烈。

📍 背景

现在大多数人用 AI 的方式是“帮我写一段代码”或“总结这篇文章”，这其实跳过了学习过程。Lathe 反其道而行之：你告诉它你想学什么（比如“Python 异步编程”），它会生成一系列学习目标、练习题、小项目，并像老师一样逐步引导你。它不替你做事，而是帮你学会做事。

为什么这事重要

  • AI 作为老师，而不是代写 — 大多数 AI 工具是“替你做”，Lathe 是“教你做”。对于 clawii 这种想提升自己技能的人来说，这比让 AI 直接写代码更有长期价值。
  • 结构化学习路径 — 不是随便问几个问题，而是生成一个从入门到精通的课程大纲，包含概念解释、代码示例、练习题和项目。你可以按自己的节奏学。
  • 开源且可定制 — 你可以修改学习路径，添加自己的资源，甚至贡献给社区。适合自学任何领域（编程、数据分析、写作等）。
🎯 今天就能做的事
  1. 安装 — pip install lathe-cli（需要 Python 3.10+ 和 OpenAI API key 或兼容的 API）。
  2. 试一个学习路径 — 运行 lathe create "用 Python 做数据分析"，它会生成一个包含 10 个模块的学习计划，每个模块有阅读材料、练习和项目。
  3. 开始学习 — 运行 lathe start 进入交互模式，它会给你出题，你回答后它给出反馈。就像有个私人 tutor。
  4. 自定义 — 编辑生成的 learning_path.yaml 文件，添加你喜欢的资源或调整难度。

跑在什么之上 — Lathe 是一个 Python CLI 工具，使用 OpenAI 兼容的 API（默认 GPT-4o，可切换 Claude 或 DeepSeek）。它通过精心设计的 prompt 让 LLM 生成结构化的学习内容，而不是直接回答。技术栈是 Python + Typer（CLI 框架）+ Jinja2（模板）。

上手难度 — 安装 2 分钟，配置 API key 30 秒。第一次生成学习路径需要 1-2 分钟（因为要调用 LLM 生成多个模块）。之后的学习过程是交互式的，每次练习需要几秒钟。

诚实说缺点 — 生成的课程质量取决于 LLM 的能力，有时会出错或太浅。需要自己判断内容准确性。目前只支持文本学习，没有视频或互动元素。对于非常专业的领域，可能不够深入。

你能拿它做什么 — 想学一个新工具（比如 n8n、Airflow）或新语言（比如 Rust、Go），让 Lathe 生成一个 2 周的学习计划，每天花 30 分钟做练习。比看教程更主动，比直接问 AI 更系统。
