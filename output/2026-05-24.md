## 👀 DeepSeek Reasonix — 跑得久成本越低的 DeepSeek 专用 agent

**EN:**

DeepSeek-Reasonix is a new open-source terminal coding agent **designed exclusively for DeepSeek models**, engineered around prefix-cache. Currently #1 on HN with 311 points. Already 5.5k GitHub stars. Real-world user reported processing 435M tokens at **$12 instead of $61** (cache hit rate: 99.82%).

**Why this matters:**
- Reasonix takes the **opposite design philosophy from opencode**: instead of supporting 75 models badly, it supports 1 model (DeepSeek) extremely well. If you're cost-sensitive, this single-vendor optimization gets you to **<$0.05 per turn**.
- For your vibe coding sessions that run for hours, the **prefix-cache magic** means each follow-up turn is nearly free. Claude Code has caching too but it's less aggressive.
- It's a leading edge of the **"sub-cent AI" era** — coding sessions that used to cost dollars are heading to cents.

**Deep dive — how is this actually different / how does it work:**

| | Reasonix | Claude Code | opencode (above) |
|---|---|---|---|
| **Model lock-in** | DeepSeek only (by design) | Claude only | 75+ models |
| **Default model** | DeepSeek-V4-Flash (cheap) | Sonnet 4.6 | Whatever you set |
| **Switch to premium** | `/pro` command → DeepSeek-V4-Pro for next turn | swap model in config | swap in config |
| **Cache strategy** | Prefix-cache obsession — 99.82% hit rate reported | Standard Anthropic caching | Depends on provider |
| **Cost per turn (typical)** | $0.05 with Flash | ~$0.10-0.50 with Sonnet | depends on provider |
| **Install** | `npm install -g reasonix` (Node.js 20.10+) | Anthropic installer | npm/brew/etc. |
| **License** | MIT | closed | MIT |

- **What it runs on**: Node.js. Talks directly to api.deepseek.com (no translation layer). Needs Node 20.10+ and a DeepSeek API key.
- **How hard to use**: `npm install -g reasonix` → get DeepSeek API key (5 min signup at platform.deepseek.com) → run `reasonix`. **10 minutes total**, similar to Claude Code.
- **What you can actually do with it**: (1) Long exploration sessions on a large codebase — cache eats the cost. (2) Anything where you don't need Claude-level reasoning — boilerplate, refactors, doc writing. (3) Run it alongside Claude Code: Reasonix for grunt work, Claude Code for hard parts. **Net result for your typical week**: probably 50-70% lower AI spend if you use it for the right tasks.
- **The honest caveat**: DeepSeek quality is good but not Claude-level for complex reasoning. The cost story is real (cache numbers verified by real users) but you'll feel the IQ drop on hard problems.

**Jargon decoder:**
- **Prefix-cache** = the agent stores the long unchanging part of every conversation (project files, instructions) so it only pays for new tokens each turn — like keeping a textbook open instead of re-buying it every page
- **API key** = your personal password for using DeepSeek's API, billable per usage
