Claude Fable 5 made permanent — stays in subscription plans

Hey clawii — Anthropic announced today that starting July 20, Claude Fable 5 will be permanently included in all Max and Team Premium plans (at 50% limits), with Pro and Team Standard users getting continued access via usage credits plus a one-time $100 credit. This marks Fable 5's shift from "limited-time trial" to "permanent standard." Behind it: competitive pressure from GPT-5.6 Sol and Kimi K3 forced Anthropic to change course.

📍 Context · Claude Fable 5 is Anthropic's strongest model, top-tier for coding, agent tasks, and multimodal work. On June 30, Anthropic planned to remove Fable 5 from subscription plans and offer it only via API, citing compute capacity needs for training. But users pushed back hard, and with OpenAI's GPT-5.6 Sol and Moonshot's Kimi K3 breathing down their necks (K3 leads Frontend Code Arena with 76% win rate), Anthropic had to reconsider. Today's permanent inclusion is a direct result of competition.

Why this matters

  • Fable 5 is no longer limited-time — you can rely on it — Previously users worried Fable 5 would vanish any day. Now you can use it stably in Max or Team Premium. For vibe coding, this is one of the best models available.
  • Competitive pressure is driving openness — GPT-5.6 Sol and Kimi K3's rise forced Anthropic to be more open. Good for users: model makers are competing to offer better value.
  • Pro users get a $100 credit subsidy — If you're not on Max, you can still use Fable 5 via credits, plus a one-time $100 bonus — essentially free usage for a while.
  • Performance benchmark — Fable 5 still leads on coding agent benchmarks, second only to K3 on Frontend Code Arena, and beats GPT-5.6 Sol and Kimi K3 on overall user experience.
