Rowboat — open-source local-first Claude Desktop alternative

Hey clawii — someone on HN posted a Show HN for a project called Rowboat. It's an open-source, local-first desktop client meant to replace Claude Desktop. Works offline, supports your own models (ollama/vLLM) or the Anthropic API. 168 pts, 51 comments. Community reception is decent.

📍 Context · More and more people want to run local models for chat, but Claude Desktop is locked to Anthropic's cloud models and requires a connection. Rowboat aims to be a local-first open alternative — data stays on your machine, you can swap in any LLM backend. Basically a pretty AI chat frontend.

Why it's worth knowing the name

  • If you're privacy-sensitive — your conversations never leave your machine. Good for sensitive stuff like internal comp data.
  • Run small models locally — hook it up to Ollama with Qwen, Gemma, GLM, etc. No API costs.
  • Can also connect to Anthropic API — you get cloud capability when you want it, with a local-first UI.
🎯 What you can do today
  1. Open the Rowboat GitHub repo and skim the README for install instructions.
  2. If you already run Ollama, give it a spin.
  3. Read the HN discussion Show HN: Rowboat to see what people are saying.
