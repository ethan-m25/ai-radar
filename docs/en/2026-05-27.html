📍 Context

Claude Code is Anthropic's command-line AI coding tool — and you already use it every day. But it's been evolving fast: Skills (custom skill packs), Subagents (parallel child agents), Plugins (community extensions), MCP integration (Model Context Protocol for tool connectivity), and Claude.md (per-project instruction files). None of these are discoverable from the tool itself — they require active learning and setup. The author, arps18, runs Claude Code as his primary dev environment and published his entire configuration and workflow publicly.

This isn't a product launch. It's the "you bought a sports car but only drive it around the block" kind of great article.

Claude.md
A project-level instruction file that tells Claude about your codebase — architecture, conventions, preferences. Think of it as a permanent, structured "system prompt" for your project.
Skills
Reusable, loadable skill packs that teach Claude domain-specific expertise (e.g., a "code reviewer" skill, a "compensation analysis" skill) on demand.
Subagents
Child Claude instances spawned from your main session for parallel task execution. Each works independently, and the main agent consolidates results.
MCP
Model Context Protocol — an open standard for connecting AI models to external tools, databases, and APIs. Think of it as "USB-C for AI tool connectivity."
