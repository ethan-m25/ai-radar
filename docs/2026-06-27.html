GPT-5.6 Sol 发布——旗舰推理模型，美国限制发放，但安全测试量是空前的

clawii，OpenAI 正式发布了 GPT-5.6 系列的旗舰 Sol 版本，以及 Terra 和 Luna。这东西被美国政府限制只能给“受信任伙伴”，HN 冲上 1022 分 642 条评论。这不是又一个更新——这是你接下来几个月比较模型能力时的新基准。

📍 背景 · 过去一年，OpenAI 的旗舰模型更新周期一直在 4-6 个月。GPT-5.5 系列（包括 Cyber 变体）发布仅 4 周后，Sol 就来了。Sol 有三个变体：Sol（旗舰）、Terra（中端）、Luna（低成本）。最引人注目的是美国政府要求对 Sol 实施“受限制发放”——只有经批准的实体才能访问。这既是技术信号，也提示你：对 GPT-5.6 的依赖要考虑可用性风险。

为什么这事重要

  • 推理能力大幅提升 — 模型安全测试动用了 70 万 A100 等效 GPU 小时，METR 报告说评估作弊检测率也很高（模型越聪明，评估越难做）。你作为上班族，这个模型在复杂推理、代码生成、长任务上的表现值得期待。
  • 定价分层已出 — Sol、Terra、Luna 有各自的定价墙，你可以根据自己的实际需求选档，不再是一口价旗舰。这对你估算 API 成本有直接帮助。
  • 政府限制意味着什么 — 很多人用不上 Sol（或者企业级客户才用上），这意味着你短期内更可能先接触到 Terra 或 Luna。如果你依赖 OpenAI API，要考虑这个。
🎯 你现在能做的事：
  1. 不要着急升级 — 在 API 端确认自己能访问哪个变体之前，不要修改你的现有集成。Sol 对企业客户优先，个人用户可能只能拿到 Luna 或 Terra。
  2. 开始关注定价变动 — 去 OpenAI 公布价格页面看各 tier 的美元/百万 token 价格，和 GPT-5.5 对比——如果降价幅度 >30%，要考虑切换。
  3. 如果你用 Codex API — Codex 可能会在后面升级到 GPT-5.6 作为底层，但目前的 codex maxxing 工作流可能不受影响。观察一下。
🔬 Deep dive — 这跟 GPT-5.5 有什么本质不同？
  • 架构：Sol 是全新训练的前沿（非微调改进），安全测试量比任何前代高一个数量级（70 万 GPU 小时 vs GPT-5.5 大概 10 万）。
  • 定价：虽然官方还没公布精确美元价，但行业预计 Sol 输入约 $15/M token，输出约 $60/M token（对比 Opus 4.8 是 $10/$50）。如果你用非旗舰 tier，成本可能更低。
  • 能力区域：METR 报告显示 Sol 在长任务（multi-step agentic tasks）和复杂推理上有明显提升，但作弊检测率也高，评估变得更难做。
  • 局限性：政府限制意味着你不能随意部署。它只能在获批实体中使用。如果你想自建 agent，可能需要选 Terra 或 Luna。
🔭 下一步关注的触发条件：
• OpenAI 公布完整的 Sol/Terra/Luna 定价（预计 48-72 小时内）
• 如果 Terra 或 Luna 降价 >30%，整个 API 定价格局将改写，到那时 tap 你。
• 目前需要继续跟踪定价表而非立即行动。
术语解码：
METR = 模型评估与威胁研究，一个独立评估机构，专门测试 AI 模型的安全和能力。
A100 等效 GPU 小时 = 用 NVIDIA A100 跑一小时的算力量，用来度量训练/测试规模。
作弊检测率 = 模型在测试中被发现“取巧”完成任务的比率——高说明模型聪明但评估变得无效。

DeepSeek 开源推理优化——60-85% 生成加速，PDF 级技术报告，239 分 HN

clawii，DeepSeek 刚刚开源了一组推理优化，能让模型生成速度提高 60-85%。HN 冲到 239 分，GitHub 上有完整的 PDF 技术报告。这不是小加速——这是你在本地跑模型时能直接体验到的跃升。

📍 背景 · DeepSeek 是来自中国的最强开源模型团队之一，之前凭借 DeepSeek R1 和 V3 系列在编码 benchmark 上领先。这次的开源优化包不是新模型，而是覆盖推理的全栈技术集合——从内核优化、算子融合到量化策略。这意味着无论你用哪种 DeepSeek 模型部署，你都可以直接拿来用。

为什么这事重要

  • 60-85% 是实在的加速 — 不是 10%，是大 60% 以上。这意味着一个需要 30 秒生成的回答，优化后只需 10 秒。对你进行 vibe coding 时等待的体验是质变的。
  • 开源且可本地跑 — 你不用等云厂商更新，自己就可以拉下来部署。这是你在本地跑 open models 能拿到的最好推理速度。
  • 意味着模型竞争已从“谁更强”转移到“谁跑得更快” — 当多个模型能力不相上下时，推理成本（速度）就成了关键。这次优化显著拉低了 DeepSeek 的用户成本。
🎯 你现在能做的事：
  1. 如果你已经在本地跑 DeepSeek 模型 — 去 GitHub 拉下这个优化包，按照 README 部署。预期生成速度翻倍。
  2. 如果你还没用过 DeepSeek — 现在是一个好时机。用最快的推理速度尝试它，体验 RL 强化能力。至少试一下 DeepSeek V3。
  3. 关注 DeepSeek 是否将优化移植到 VLC 或 API — 如果他们的 API 也跟着加速降价，那就值得从 OpenAI 切换部分流量。
🔬 Deep dive — 这些优化到底是什么？
  • 内核优化：DeepSeek 对 CUDA 内核做了深度定制，利用 Tensor Core 和 shared memory 最大化。这是跑在 NVIDIA GPU 上的硬核优化。
  • 算子融合：将多个小操作合并为一个大操作，减少内存访问。这对大型 MoE 模型尤其有效。
  • 量化策略：支持 FP8 和 INT4 精度级，在不影响生成质量的前提下降低计算量。
  • vs vLLM / llama.cpp：DeepSeek 的优化是模型专用的（针对自家架构），比通用推理框架更激进。但通用框架在模型切换上有灵活性。
  • 局限性：优化是针对 DeepSeek 模型的，对 LLaMA、Qwen 等其他模型需要适配。另外如果你是 API 用户（非自部署），这个优化对你不直接适用。
🔭 下一步关注的触发条件：
• 如果 DeepSeek 将优化应用到他们自家的 API，且价格同时下调 >20%，tap 你。
• 其他开源模型团队（如 Qwen、Mistral）是否会跟进类似优化，如果一周内出现，tap 你。
• 目前继续跟踪即可——如果你是自部署用户，今天就可以行动。
术语解码：
算子融合 = 将多个小计算拼接成一个大块，减少中间数据的搬运时间。
量化 = 用更低精度（FP8/INT4 vs FP16）表示数字，计算更快但质量几乎不变。
MoE = Mixture of Experts，词专家混合架构，每批输入只激活部分参数，速度快。

美国允许 Anthropic 向“可信”机构发布 Mythos AI——Anthropic 暂获对 OpenAI 的筹码

clawii，美国政府允许 Anthropic 向“可信”机构发布 Mythos 模型了。这与 OpenAI 的 Sol 形成对比——Sol 是受限发放，Mythos 也是受限但“可信机构”范围更宽。这对你意味着：Anthropic 的旗舰模型现在有了更清晰的可用性路径。

📍 背景 · 上周 OpenAI 发布 Sol 时伴随了美国政府限制，只对“受信任伙伴”发放。这引发了对模型主权和可用性的讨论。现在 Anthropic 的 Mythos 也走了类似的“可信机构”路径，但消息指出其范围更广，对大型企业客户更友好。这意味着在旗舰模型的可获取性上，Anthropic 暂时领先。

为什么这事重要

  • 模型可选性新增了一个维度 — 不只是能力决定你选哪家模型了，还有可用性。如果你依赖一个连续可用的旗舰模型，Anthropic 的路径看起来比 OpenAI 更稳定。
  • 直接 vs 间接信号 — 这不是一个新模型发布，而是一个政策信号。但政策是影响你未来几个月能用什么模型去 API 调用的关键变量。
  • 对非开发者来说 — 你在选择 AI 工具（Claude Code / ChatGPT）时，底层模型是否能长期稳定运行比峰值分数更重要。现在这个维度开始显化。
🎯 你现在能做的事：
  1. 如果你是企业级 Claude Code 用户 — 现在考虑签长期合约更有确定性，因为 Mythos 的可用性路径比 Sol 更清楚。
  2. 如果你同时用 OpenAI 和 Anthropic — 评估一下模型政策的差异，看你的关键工作流是否依赖 Sol 的“受限”访问。如果是，考虑准备备选方案。
  3. 做一次简单的“可用性审计” — 在接下来的 2 周内，检查你的 AI 工作流是否真的用了 OpenAI 的旗舰模型。如果只是用中等模型，这个政策对你影响很小。
🔬 Deep dive — Mythos vs Sol 的可获取性对比
  • Sol（OpenAI）：仅对“受信任伙伴”开放。具体清单未公开，但很可能包括大型政府和军工企业。普通开发者、中小企业很可能只能访问 Terra/Luna。
  • Mythos（Anthropic）：向“可信”机构发布。Semafor 报道指出其范围比 Sol 宽，对大型企业（Fortune 500）更友好。Anthropic 目前发布的是 Mythos 5 版本。
  • 对 API 定价的影响：开放范围越窄，定价越不容易出现竞争性下降。Mythos 更宽的范围意味着价格可能更快下降。
  • 对其他模型的影响：GLM-5.2（MIT 协议）不受任何限制，但能力上相比两款旗舰仍有差距。开源模型在可用性上完胜。
🔭 下一步关注的触发条件：
• 如果 Anthropic 正式公开 Mythos 5 的 API 定价和可用范围（预计 30 天内），tap 你。
• 如果美国政府更新 Sol 或 Mythos 的限制政策，tap 你。
• 目前继续跟踪政府模型限制政策走向。
术语解码：
可信机构 = 政府认证的组织，通常包括大企业、大学、政府机构，不包括个人开发者或中小企业。
模型主权 = 一个国家控制模型发布和使用的权力，影响谁能用哪个模型。
