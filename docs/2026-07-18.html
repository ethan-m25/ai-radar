Claude Fable 5 保留在订阅计划中

爪哩，Anthropic 今天宣布，从 7 月 20 日起，Claude Fable 5 将包含在 Max 和 Team Premium 订阅计划中，限额为 50%。Pro 和 Team Standard 用户仍可通过积分使用，并会获得一次性 $100 积分。这是对之前 Fable 5 只向 API 收费的计划的大反转。Simon Willison 今天写了博客，称其为 "Fablepocalypse 的结束"。

📍 背景 · Claude Fable 5 是 Anthropic 目前最强的模型，在编码和 agent 任务上表现极佳。6 月 30 日，Anthropic 曾短暂移除 Fable 5，但在 7 月 1 日恢复访问，同时宣布未来将只通过 API 收费，不包含在订阅中。这一决定引发了大量用户不满，因为 Max 订阅每月 $200，如果没有 Fable 5 就失去了很大价值。现在 Anthropic 改变了主意，将 Fable 5 保留在订阅中，但限额 50%。这可能是迫于竞争压力——GPT-5.6 Sol 和 Kimi K3 都在追赶。

为什么这事重要

  • Max 订阅用户终于能安心了 — 如果你每月花 $200 订阅 Max，现在可以继续使用 Fable 5，而不用额外支付 API 费用。虽然限额 50%，但总比没有好。
  • 竞争压力改变了 Anthropic 的策略 — GPT-5.6 Sol 和 Kimi K3 的发布让 Anthropic 意识到，如果只通过 API 收费，用户可能会流向其他平台。保留 Fable 5 在订阅中是一个竞争性决策。
  • 对 Pro 用户也有利好 — Pro 和 Team Standard 用户会获得 $100 积分，可以继续使用 Fable 5 的 API。
🎯 今天就能做的事
  1. 如果你有 Max 订阅，7 月 20 日后你的 Fable 5 限额将自动恢复。无需任何操作。
  2. 如果你是 Pro 用户，检查你的账户是否收到 $100 积分。如果没有，联系 Anthropic 支持。
  3. 在 Fable 5 上测试你最重要的编码任务——确保模型能力仍然满足你的需求。
