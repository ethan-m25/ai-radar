🚨 Must See · 第 1 周信号

opencode — 一个野生 Claude Code，能接 75+ 种模型

GitHub 涨势最猛的 AI 项目，把 Claude Code 和 codex 都甩在身后。MIT 开源，能在同一个工具里换不同公司的模型用。你用过 Claude Code 的话，5 分钟就能上手。

📍 背景 · Context

编程 agent 这个赛道过去 18 个月一直是 Claude Code（Anthropic 闭源）和 codex（OpenAI 闭源）两家独大。社区想要的"开源 + 模型自由"一直没有真正能打的方案——直到 opencode。它是第一个在 GitHub 上跑赢两个闭源大哥的开源对手，而且不是昙花一现：28 天 +1540 star 的涨幅是 Claude Code 同期的 1.9 倍。

Why this matters

  • This is the first credible open-source competitor to Claude Code / codex with real traction — not hype. Independent dev teams are voting with their stars.
  • You can mix-and-match models inside the same workflow — cheap DeepSeek for boilerplate, Claude Opus 4.7 for tricky logic, local Ollama for sensitive code, all in one tool with one set of muscle memory.
  • The architecture is client-server, meaning future remote-pair-programming patterns become possible — something neither Claude Code nor codex offers today.
  • You're catching this in week 1 of breakout — exactly the SLA we set. Red Note / 微信视频 won't cover this until late June. You have a 4-6 week head start to learn it.
🎯 实操 — 你今天能做什么
  1. 装一下试试：brew install opencode 或 npm install -g opencode，5 分钟。
  2. 把你的 DeepSeek API key 塞进去，跑一段你平时用 Claude Code 写的代码——感受一下质量差距和成本差距。
  3. 如果觉得不错，下次 vibe coding 时把它和 Claude Code 并排开两个窗口，分流任务（简单的给 opencode，难的给 Claude Code）。

跑在什么之上：TypeScript 写的（占 66%）。自带 API key，想用哪家模型用哪家，不绑定厂商。

上手难度：brew install opencode → 设环境变量塞 API key → 终端敲 opencode。用过 Claude Code 的话 5 分钟搞定，界面很像。

诚实说明：我没亲自验证它的体验是不是和 Claude Code 一样顺。能接 75 种模型也可能意味着"什么都能但什么都不精"。涨 star 说明有人喜欢，但只有你自己花 30 分钟试一下才知道你喜不喜欢。

👀 Worth a Look · 成本拐点

DeepSeek Reasonix — 缓存命中率 99.82%，把成本砍到 1/5

DeepSeek 自家出的专用编程 agent，跑越久越省。真实用户报告：4.35 亿 token 一天从 $61 降到 $12。HN 头条第 1。走的是和 opencode 完全相反的路。

📍 背景 · Context

DeepSeek 同一周还宣布旗舰模型永久降价 75%（Bloomberg, HN 163 分）——意味着 AI 推理正在变成大宗商品，价格往零冲。Reasonix 是这个战略的产品层落地：不只是模型便宜，还要让 agent 工具本身把缓存玩到极致，让"长时间挂着 agent 跑"的成本接近免费。这是过去半年最重要的成本拐点之一。

Why this matters

  • Opposite philosophy from opencode: instead of supporting 75 models okay, support 1 model (DeepSeek) extremely well. Single-vendor optimization → under $0.05 per turn.
  • Long vibe-coding sessions (the way you actually use Claude Code) benefit most — prefix-cache magic means each follow-up turn is nearly free. Claude Code caches too but less aggressively.
  • Combined with DeepSeek's same-week 75% permanent price cut, this is the leading edge of the "sub-cent AI" era — coding sessions that used to cost dollars are heading to cents.
  • Real users in the wild are already reporting 5× cost reductions — verified, not theoretical.
🎯 实操 — 你今天能做什么
  1. 装：npm install -g reasonix（需要 Node 20.10+），10 分钟搞定。
  2. 去 platform.deepseek.com 注册拿 API key（5 分钟）。
  3. 下次需要做"重复性、不需要 Claude 级智商"的任务时——比如批量整理数据、读长文档、写样板代码——切到 Reasonix。Claude Code 留给真正的硬骨头。
  4. 跑一周看看你的 AI 花费实际降了多少。预期：split 用得当的话省 50-70%。

👀 Worth a Look · 企业 AI 拐点

KPMG 把 Claude 部署给了 27.6 万员工

迄今最大规模的企业 AI 部署之一。四大对生成式 AI 的犹豫期正式结束。你公司的 IT/法务/合规 30 天内会面对同样的对话——你不应该是最后一个知道的人。

📍 背景 · Context

四大会计师事务所（KPMG、德勤、EY、普华永道）是企业里风险偏好最保守的群体——他们做审计和合规咨询的钱靠的就是"绝对靠谱"的品牌。这种公司过去两年对生成式 AI 一直是"小心翼翼试点"的姿态。KPMG 这次直接铺到全球 27.6 万员工，意味着内部已经过了所有审计/法务/数据隐私的审查。这是一个分水岭信号：其他三大现在被动反应，年底之前几乎肯定跟进。

EN

Why this matters

  • You explicitly said you wanted to know "what big companies are doing with AI" — this is one of the strongest enterprise signals of 2026. KPMG has the most conservative risk profile of anyone you'd compare to.
  • For your day job: this kind of move hits C-suites of every other Big-4 within 30 days. Your multinational's IT/Legal/Compliance teams will face the same conversation. Knowing the trend ahead means you can frame it instead of reacting.
  • For HR / Compensation specifically: these are early enterprise AI use cases (JD writing, benchmark analysis, policy drafting). Once your IT formally deploys Claude (~6-12 months out), being already-fluent in personal use means you ramp into the official workflow on day 1.
  • The choice of Anthropic over OpenAI is itself a signal — Claude's "constitutional AI" framing reads better to audit-firm compliance teams. Expect more audit/legal/finance firms to follow the same pattern.
🎯 实操 — 你今天能做什么
  1. 个人层面：保持你已经在做的——用 Claude/Claude Code 处理薪酬政策草稿、JD 写作、salary benchmark 分析、读长 HR/劳动法规文件。你已经在做了，继续做。
  2. 组织层面（如果你想推动）：找一个 HR/薪酬里又痛又标准的流程，试点 Claude Pro 或 Team 套餐（$20-30/人/月），量化省下来的时间，写一页纸提案。KPMG 这条新闻是 PPT 的弹药。
  3. 对话层面：下次和 IT/法务同事聊 AI 时，提一下"KPMG 已经全员部署 Claude 了"——把你定位成"知道行业进度的人"。
